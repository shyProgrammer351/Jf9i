문제

돈을 입력 받아, 500원, 100원, 50원, 10원의 개수를 계산하는 문제입니다 실행순서를 클릭하세요 
잔돈을 입력하세요: 3210
3210원을 500원짜리 동전 6개, 100원짜리 동전 2개, 50원짜리 동전 0개, 10원짜리 동전 1개로 거슬러 드립니다.

	#include <stdlib.h>

		int _500;

2		return payment;

	}

4		Coin* coin =

6		return coin;

	}

			payment, coin->_500, coin->_100, coin->_50, coin->_10 );
