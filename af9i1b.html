문제

파일 원본을 읽어 파일 복사본을 생성하는 파일 복사 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
--------- 파일 원본.txt의 내용 --------- 
과연 파일 복사는 잘 될까요?
파일 복사가 잘 되면 좋겠어요^^

코드 빈칸 채우기

#include <stdio.h> 
 
#define MAX_STR_LEN 1024	 
 
int main() 
{ 
	char line[MAX_STR_LEN] = ""; 
 
	inFile = ( "원본.txt", "" );			 
	outFile = ( "복사본.txt", "" );			 
	if ( ( inFile == NULL ) || ( outFile == NULL ) )		 
	{ 
		printf( "[오류] 파일을 열 수 없습니다!" ); 
		return -1; 
	} 
 
	while ( ( line, MAX_STR_LEN, inFile ) != NULL ) 	 
	{ 
		( outFile, "%s", line ); 			 
	} 
 
	( inFile );						 
	( outFile );							 
 
	return 0; 
}
