문제

영어 단어를 입력하면 영한 사전을 열고 그 뜻을 찾아서 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
--------- 파일 dictionary.txt의 내용 --------- 
apple의 뜻은 사과입니다.
banana의 뜻은 바나나입니다.
carrot의 뜻은 당근입니다.
...

영어단어를 입력하세요 : apple
apple의 뜻은 사과입니다.


영어단어를 입력하세요 : banana
banana의 뜻은 바나나입니다.


영어단어를 입력하세요 : carrot
carrot의 뜻은 당근입니다.

알고리즘

프로그램시작 
   영어 단어를 입력 
   영한 사전을 열기 
   영한 사전에서 한 줄씩 읽고 
      해당 영어단어를 찾으면 뜻을 출력 
   영한 사전 닫기 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter09/Dictionary.c 

	#include <stdio.h> 

	#include <string.h> 

	 

	// 문자열 최대 길이는 1024 

	#define MAX_STR_LEN 1024	 

 

	// 프로그램시작 

1	int main() 

	{ 

		char word[MAX_STR_LEN] = ""; 

		char meaning[MAX_STR_LEN] = ""; 

2		FILE *dictionary = NULL;						 

	 

		// 영어 단어를 입력 

		printf( "영어단어를 입력하세요 : " ); 

		// banana를 입력한다고 가정 

3		scanf( "%s", word ); 

 

		// 영한 사전을 열기 

4		if ( ( dictionary = fopen( "dictionary.txt", "rt" ) ) == NULL )		 

		{ 

			printf( "[오류] 파일을 열 수 없습니다!" ); 

5			return -1; 

		} 

	 

		// 영한 사전에서 한 줄씩 읽고 

6		while ( fgets( meaning, MAX_STR_LEN, dictionary ) != NULL ) 		 

		{ 

			// 해당 영어단어를 찾으면 뜻을 출력 

7			if ( !strncmp( word,  meaning, strlen( word ) ) ) 

			{ 

				printf( "%s", meaning ); 

8				break; 

			} 

		} 

 

		// 영한 사전 닫기 

9		fclose( dictionary );							 

	 

		// 프로그램 종료 

10		return 0; 

	}
