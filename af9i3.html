문제

돈을 입력 받아, 500원, 100원, 50원, 10원의 개수를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
잔돈을 입력하세요: 3210
3210원을 500원짜리 동전 6개, 100원짜리 동전 2개, 50원짜리 동전 0개, 10원짜리 동전 1개로 거슬러 드립니다.


잔돈을 입력하세요: 1230
1230원을 500원짜리 동전 2개, 100원짜리 동전 2개, 50원짜리 동전 0개, 10원짜리 동전 3개로 거슬러 드립니다.


잔돈을 입력하세요: 100
100원을 500원짜리 동전 0개, 100원짜리 동전 1개, 50원짜리 동전 0개, 10원짜리 동전 0개로 거슬러 드립니다.

알고리즘

 프로그램 시작 
    사용자에게 돈을 입력받기 
    동전 개수를 계산 
    동전 개수를 출력 
 프로그램 종료 
 
 돈을 입력받는 함수 시작 
    사용자에게 돈을 입력받기 
 돈을 입력받는 함수 종료 
 
 동전 개수를 계산하는 함수 시작 
    잔액에서 500원짜리 동전 개수 계산하고, 잔액 재조정 
    잔액에서 100원짜리 동전 개수 계산하고, 잔액 재조정 
    잔액에서 50원짜리 동전 개수 계산하고, 잔액 재조정 
    잔액에서 10원짜리 동전 개수 계산하고, 잔액 재조정 
 동전 개수를 계산하는 함수 종료

프로그램 코드 

	// 파일명 : ./Chapter09/Coin.c 

	#include <stdio.h> 

	#include <stdlib.h> 

	 

	typedef struct _Coin { 					 

		int _500; 

		int _100; 

		int _50; 

		int _10; 

	} Coin;     

	 

	// 돈을 입력받는 함수 시작 

1	int getPayment() 

	{ 

		int payment = 0; 

	 

		// 사용자에게 돈을 입력받기 

		printf( "잔돈을 입력하세요: " ); 

		scanf( "%d", &payment ); 						 

	 

		// 돈을 입력받는 함수 종료 

2		return payment; 

	} 

	 

	// 동전 개수를 계산하는 함수 시작 

3	Coin* getCoins( int payment )			 

	{ 

		// 동전 변수 동적 할당 

4		Coin* coin = 						 

5			     (Coin*)malloc( sizeof(Coin) );		 

	 

		// 잔액에서 500원짜리 동전 개수 계산하고, 잔액 재조정 

		coin->_500 = payment / 500;							 

		payment = payment % 500;						 

	 

		// 잔액에서 100원짜리 동전 개수 계산하고, 잔액 재조정 

		coin->_100 = payment / 100;							 

		payment = payment % 100;					 

	 

		// 잔액에서 50원짜리 동전 개수 계산하고, 잔액 재조정 

		coin->_50  = payment / 50;							 

		payment = payment % 50;						 

	 

		// 잔액에서 10원짜리 동전 개수 계산하고, 잔액 재조정 

		coin->_10  = payment / 10;							 

		payment = payment % 10;					 

	 

		// 동전 개수를 계산하는 함수 종료 

6		return coin; 

	} 

	 

	// 프로그램 시작 

7	int main() 

	{ 

		int payment = 0;						 

		Coin* coin = NULL;								 

	 

		// 사용자에게 돈을 입력받기 

8		payment = 

9			  getPayment(); 

	 

		// 동전 개수를 계산 

10		coin =  

11		        getCoins( payment ); 

	 

		// 동전 개수를 출력 

12		printf( "%d원을 500원짜리동전 %d개,100원짜리동전 %d개,50원짜리동전 %d개,10원짜리동전 %d개로 거슬러 드립니다.",  

			payment, coin->_500, coin->_100, coin->_50, coin->_10 );		 

	 

		// 동적 할당 받은 동전 변수 반납 

13		free( coin );									 

	 

		// 프로그램 종료 

14		return 0; 

	}
