문제

입력받은 달의 영어이름을 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
1월 ~ 12월 사이의 월을 입력하세요: 4
4월은 April입니다

프로그램 코드

	import java.util.Scanner;
	public class Month
	{  
1		public static void main( String[] args ) {  
			Scanner scan = new Scanner( System.in );																
2			System.out.print( "1월 ~ 12월 사이의 월을 입력하세요: " );
3			int month = scan.nextInt();
			
4			String[] monthName = { "January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November",  "December" }; 
5			System.out.print( month + "월은 " + monthName[ month - 1 ] + "입니다" );
		
			scan.close();
6		}																					
	}
