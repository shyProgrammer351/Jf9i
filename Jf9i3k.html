 단어나 문장을 저장하기 위한 문자 배열  

 배열의 여러 칸 중에서 첫번째 칸을 나타내는 번호  

 배열의 길이  

 여는 대괄호로 특정 배열 원소를 가리킬 때 사용하는 기호  

 닫는 대괄호로 특정 배열 원소를 가리킬 때 사용하는 기호  

 증가연산자로 연산자의 왼쪽 변수값에 1만큼 더한 값을 왼쪽 변수 공간에 저장  

 감소연산자로 연산자의 왼쪽 변수값에 1만큼 뺀 값을 왼쪽 변수 공간에 저장  

 비교연산자로 왼쪽 값이 오른쪽 값보다 크면 참을 반환하고 아니면 거짓을 반환  

 비교연산자로 왼쪽 값이 오른쪽 값보다 작으면 참을 반환하고 아니면 거짓을 반환  

 비교연산자로 왼쪽 값이 오른쪽 값보다 작거나 같으면 참을 반환하고 아니면 거짓을 반환  

 클래스 객체의 데이터 값과 전달인자의 데이터 값을 비교한 차이를 반환하며, 같으면 0을 반환  

 문자열 데이터를 한 단어씩 읽어오는 메서드  

 초기식, 조건식, 증감식으로 구성되고, 조건식이 참일 때마다 해당 명령문을 실행하는 반복문  

 조건식이 참이면 해당 명령문을 실행하는 선택문
